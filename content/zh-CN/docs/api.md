# Jannchie API 参考

## 可用端点

- `GET /api/health` — 健康检查
- `GET /api/_sitemap-urls` — 站点地图 URL
- `GET /llms.txt` — AI Agent 用站点摘要
- `GET /llms-full.txt` — 完整文档
- `GET /openapi.json` — OpenAPI 3.1.0 规范
